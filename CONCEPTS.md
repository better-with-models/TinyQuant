# Concepts

## Compiled knowledge layer

The compiled knowledge layer is the `docs/` directory. It is the maintained
Obsidian wiki where durable summaries, concepts, entities, specs, and design
notes live.

See also: [docs/README.md](docs/README.md), [AGENTS.md](AGENTS.md)

## Raw evidence layer

The raw evidence layer includes `docs/research/` and the rest of the
repository outside `docs/` when those files are being analyzed as source
material for the wiki.

See also: [docs/research/llm-wiki.md](docs/research/llm-wiki.md),
[AGENTS.md](AGENTS.md)

## Obsidian markdown boundary

Obsidian-specific markdown features such as wikilinks, embeds, and callouts are
allowed inside `docs/` and should not appear in markdown files outside `docs/`.

See also: [README.md](README.md), [AGENTS.md](AGENTS.md)
