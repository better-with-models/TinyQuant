# src

TypeScript wrapper modules that present the native `@tinyquant/core` API.

## Files

| File | Purpose |
| --- | --- |
| `index.ts` | Public barrel for package exports |
| `_loader.ts` | Platform loader for bundled native binaries |
| `_errors.ts` | Error parsing and `TinyQuantError` helpers |
| `backend.ts` | Brute-force backend wrappers |
| `codec.ts` | Codec-related wrappers |
| `corpus.ts` | Corpus and policy wrappers |
