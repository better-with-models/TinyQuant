# commands

Per-subcommand command handlers for the `tinyquant` CLI.

## Files

| File | Purpose |
| --- | --- |
| `codec.rs` | Shared codec command wiring |
| `codec_train.rs` | Codebook training command |
| `codec_compress.rs` | Compression command |
| `codec_decompress.rs` | Decompression command |
| `codebook_io.rs` | Codebook import and export helpers |
| `corpus.rs` | Shared corpus command wiring |
| `corpus_ingest.rs` | Corpus ingest command |
| `corpus_search.rs` | Corpus search command |
| `info.rs` | Build and environment reporting |
| `verify.rs` | Verification command |
| `mod.rs` | Module registration |
