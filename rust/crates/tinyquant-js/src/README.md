# src

Rust `napi-rs` source for the `@tinyquant/core` native binding.

## Files

| File | Purpose |
| --- | --- |
| `lib.rs` | Module root and napi registration |
| `codec.rs` | Codec binding types and methods |
| `corpus.rs` | Corpus binding types and methods |
| `backend.rs` | Search backend bindings |
| `buffers.rs` | Typed-buffer helpers |
| `errors.rs` | Error conversion and reporting |
